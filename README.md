## ✨ Features

- **Centralized Management** — Single admin panel for all services
- **Portainer Integration** — Built-in container management
- **Privacy-Focused** — Self-hosted search with SearXNG
- **Vector Storage** — Qdrant for long-term memory
- **Browser Automation** — CloakBrowser for agent tasks
- **Obsidian Integration** — Note-taking and knowledge management
- **Tailscale Networking** — Secure connectivity out of the box

## 🚀 Quick Start

```bash
git clone https://github.com/OneByJorah/StackDeploy-Dashboard.git
cd StackDeploy-Dashboard
sudo ./bootstrap.sh
```

## 🏗️ Services

| Service | Description |
|---------|-------------|
| SearXNG | Private meta-search engine |
| Camofox | Privacy-focused rendering |
| CloakBrowser | Browser automation |
| Qdrant | Vector database |
